### Wiring among Arduino, Raspberry Pi and PC. 
```
Universal Asynchronous Reception and Transmission (UART)
All Arduinos                   RPi 
5V  Pin  <--Jump-Wire-->    5V Pin
GND Pin  <--Jump-Wire-->   GND Pin
TX  Pin  <--Jump-Wire-->    RX Pin
RX  Pin  <--Jump-Wire-->    TX Pin
```
