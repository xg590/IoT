### D1 mini
* VCP driver: CH340
* LOLIN(WEMOS) D1 mini(clone)
* Pin Voltage: 3.3v
* The ESP8266 supports interrupts in any GPIO, except GPIO16.
* Pinout [More Info](https://randomnerdtutorials.com/esp8266-pinout-reference-gpios/)<br/>
