# How to Use
* Board
  * Raspberry Pi: ZW / 3A / 3B / 4
  * Arduino: Uno / Nano / Pro Mini
  * ESP8266 
  * ESP32
* Communication
  * LoRa: 
  * Radio
  * I2C, SPI, UART
  * Infra-red
* Motion
  * Motor: DRV8833 / PCA9685+TB6612 
  * Servo
  * Relay
  * MOSFET
  * Rotary Encoder
* Sensor
  * GPS: NEO-6M
  * Compass: BNO055 / MPU-6050
